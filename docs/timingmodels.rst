Timing Models
=============

This section is to explain the bulit-in timing models.

PINT built-in timing models are located at the directory PINT/src/pint/models_.
Timing model is structured as model components which implement difference pieces
of timing model, and the parent ``TimingModel`` object for overall model components
storage and managing.
